Compatibilité

Discord offre des applications de bureau. Ces dernières sont compatibles avec la grande majorité des systèmes d’exploitation connus, c’est-à-dire :

  • Windows
  • macOS (anciennement OS X, ce nom est à oublier aujourd’hui)
  • Linux
  • iOS
  • iPadOS
  • Android

Cependant, un grand avantage de discord est qu’il peut être ouvert dans n’importe quel navigateur récent !
