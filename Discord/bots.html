Bots

Discord offre une api accessible par tous.

L'api permet de créer des « Bots » qui prennent la forme d’utilisateurs. On peut ensuite programmer ces « utilisateurs » pour faire différentes actions, le tout dans un programme évènementiel.

Des contributeurs bénévoles se sont rassemblés et ont créé plusieurs bibliothèques pour différents langages de programmation, afin d’accéder à cette api dans son langage préféré !
